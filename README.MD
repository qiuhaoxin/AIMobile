## AIMobile 代码项目结构
   src 源码结构
   |
   |---lib 存放第三方库 
   |
   |---public 
   |
   |---services 网络请求
   |
   |---pages 存放各个页面

   |
   |---routes 页面之间的路由
   |
   |---reducers redux 的state管理
   |
   |---lib 第三方依赖库
   |
   |---components 页面组件
   |
   |---action  包括actionType  和 请求数据方法
   |
   |---images 图片资源
